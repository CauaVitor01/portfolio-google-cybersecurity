## Projetos e Documentação Técnica



Abaixo estão os principais projetos práticos desenvolvidos durante minha formação:



### Automação e Programação

* **[Algoritmo de Atualização de Arquivos em Python](./Algoritmo%20de%20Atualização%20de%20Arquivos%20em%20Python.pdf):** Criação de um script para automatizar o controle de acesso (allow list), removendo IPs não autorizados de arquivos de sistema.

* **[Filtros e Consultas SQL](./Aplicar%20filtros%20a%20consultas%20SQL.pdf):** Aplicação de filtros avançados para identificar tentativas de login suspeitas e gerenciar bancos de dados de funcionários.



### Resposta a Incidentes e Análise

* **[Análise de Incidente DDoS (NIST CSF)](./Análise%20de%20relatórios%20de%20incidentes.pdf):** Documentação completa de um ataque ICMP Flood, aplicando as fases *Identify, Protect, Detect, Respond* e *Recover*.

* **[Diário de Incidentes](./Diário%20do%20responsável%20pelo%20tratamento%20de%20incidentes.pdf):** Relatórios detalhados sobre ataques de Ransomware e falhas de IDOR, utilizando a metodologia dos "5 Porquês".



### Governança, Riscos e Conformidade (GRC)

* **[Avaliação de Vulnerabilidades](./Relatório%20de%20avaliação%20de%20vulnerabilidades.pdf):** Relatório técnico focado em segurança de bancos de dados MySQL e riscos de Injeção SQL.

* **[Auditoria de Controles e Conformidade](./Lista%20de%20verificação%20de%20controles%20e%20conformidade%20.pdf):** Análise de conformidade baseada em frameworks como PCI-DSS, GDPR e SOC, com recomendações de melhoria.
